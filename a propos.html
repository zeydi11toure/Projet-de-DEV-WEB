Conseils de l'équipe chargée des relations avec les développeurs des etudiant de ig l2

Nous souhaitons vous aider à créer des sites Web attrayants, accessibles, rapides et sécurisés, compatibles avec tous les navigateurs. Sur ce site, vous trouverez du contenu rédigé par des membres de l'équipe Chrome et des experts externes pour vous aider dans votre parcours.

Sur ce site

Parcourez nos archives, qui couvrent tous les sujets liés à la pile de développement Web. Consultez les dernières actualités et contenus sur notre blog, consultez les derniers articles et découvrez des contenus par groupes sur la page Explorer.

Pour accéder à des cours complets sur les principaux sujets liés au développement Web, consultez la page Learn (Apprendre). Développés par des experts du secteur, ces cours vous guident dans un sujet précis. Leur structure vous permet de vous plonger dans un ou deux modules qui vous intéressent.

Vous voulez du code à emporter ? Nos modèles détaillent les exigences courantes de développement Web. Que vous souhaitiez comprendre comment utiliser un certain composant d'interface utilisateur ou savoir comment copier une image dans le presse-papiers, vous trouverez la solution adaptée.

Pour vous aider à justifier votre travail sur l'interface utilisateur ou les performances, consultez nos études de cas. Découvrez comment d'autres entreprises ont utilisé des métriques telles que les Core Web Vitals pour obtenir des résultats concrets. Et si cela vous suffit pour une journée, nous disposons également d'une excellente archive de podcasts et émissions.

Nous pensons qu'il est important de disposer de plusieurs moteurs de navigation sur le Web et que les sites Web et les applications doivent fonctionner correctement quel que soit le navigateur utilisé par vos visiteurs. Nous savons que cela vous intéresse également, c'est pourquoi le contenu de ce site doit présenter l'état multi-navigateurs clairement. Comme vous pourrez le constater, de nombreux articles contiennent un composant indiquant la prise en charge des navigateurs. Les données proviennent de nos amis de MDN, via le projet "Browser Compat Data" qui alimente les données des pages MDN.

Améliorer le Web pour demain

Nous espérons que web.dev vous aidera à créer des expériences Web de haute qualité dès aujourd'hui. L'équipe ig l2 s'efforce également d'améliorer le Web demain. Si vous souhaitez en savoir plus sur ce que nous faisons, et si vous souhaitez nous faire part de vos commentaires sur les fonctionnalités que nous développons et auxquelles nous contribuons, consultez la page ig l2 pour les développeurs.
